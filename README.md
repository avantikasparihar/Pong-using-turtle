# Pong-using-turtle

Classic Pong game using turtle module of Python v 3.8.2

Pong is a table tennis sports game featuring simple two-dimensional graphics, manufactured by Atari and originally released in 1972.
It was one of the earliest arcade video games and the first commercially successful video game.

This classic version of the Pong game has been built using turtle graphics module of Python, simple game logic and audio for added effects.
